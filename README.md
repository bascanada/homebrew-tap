# Homebrew Tap for BasCanada

This repository contains Homebrew formulae for BasCanada projects.

## Usage

To install a package from this tap:

```bash
brew tap bascanada/tap
brew install <package-name>
```

## Packages

- **logviewer**: A log viewer utility.
- **harmonium**: A cli/lib to generate procedural music


## Casks

- **harmonium-plugins**: VST/Clap plugin for harmonium
